Flex Container Properties

display

flex | inline-flex

display: flex;

flex-direction

row | column | reverse

flex-wrap

nowrap | wrap | reverse

flex-wrap: wrap;

flex-flow

Shorthand of direction & wrap

flex-flow: row wrap;

justify-content

Main axis alignment

justify-content: space-between;

align-items

Cross axis alignment

align-items: center;

align-content

Multi-line alignment

align-content: stretch;

Flex Item Properties

order

Controls item order

order: 1;

flex-grow

Allow item to grow

flex-grow: 1;

flex-shrink

Allow item to shrink

flex-shrink: 1;

flex-basis

Initial size of item

flex-basis: 100px;

flex

Shorthand for grow/shrink/basis

flex: 1 1 auto;

align-self

Override cross alignment

align-self: flex-end;
