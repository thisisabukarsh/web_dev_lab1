CSS: Three Ways to Style

1

Inline CSS

Applied directly to individual HTML elements

<p style="color: blue; font-size: 16px; margin: 10px;"> This is blue text </p>
Quick for single elements
Not reusable
Hard to maintain
2

Internal CSS

Defined in the head section of HTML

<style> p { color: blue; font-size: 16px; } h1 { color: red; } </style>
Works for one page
Reusable within page
Not for multiple pages
3

External CSS

Separate CSS file linked to HTML

/* In HTML file */ <link rel="stylesheet" href="styles.css"> /* In styles.css */ p { color: blue; }
Reusable across site
Better organization
Faster page loading

Best Practice: Use external CSS files for maintainable, scalable websites
