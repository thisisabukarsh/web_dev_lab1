Grid Container & Item Properties

Below is a categorized list of all CSS Grid properties . Mini-Game

Grid Container Properties

display

grid | inline-grid

display: grid;

grid-template-columns

Defines column structure

grid-template-columns: 1fr 2fr;

grid-template-rows

Defines row structure

grid-template-rows: auto 100px;

grid-template

Shorthand for rows/columns

grid-template: auto / 1fr 1fr;

gap

Sets row and column gaps

gap: 10px;

justify-items

Aligns items horizontally

justify-items: center;

align-items

Aligns items vertically

align-items: stretch;

Grid Item Properties

grid-column

Item's column position

grid-column: 1 / 3;

grid-row

Item's row position

grid-row: 2 / 4;

grid-area

Shorthand for row/column

grid-area: 1 / 1 / 3 / 3;

justify-self

Aligns item horizontally

justify-self: end;

align-self

Aligns item vertically

align-self: start;
