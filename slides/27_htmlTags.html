Most Important HTML Tags

Essential building blocks for every web project : check this

Structural Tags

<html>

Root of the HTML document

<html lang="en"></html>

<head>

Metadata, links, title

<head><title>Page</title></head>

<body>

Main content container

<body>...</body>

<h1> - <h6>

Headings levels 1 to 6

<h1>Title</h1>

<p>

Paragraph of text

<p>Hello World</p>

Grouping & Layout

<div>

Generic container for layout

<div>Content</div>

<span>

Inline container

<span>Text</span>

<img>

Embeds images

<img src="image.jpg" alt="desc" />

<a>

Hyperlinks to other pages

<a href="#">Link</a>

<ul> / <ol> / <li>

Lists: unordered, ordered, list items

<ul><li>Item</li></ul>

Forms & Inputs

<form>

Form container

<form action="#">...</form>

<input>

Input fields

<input type="text" placeholder="Enter" />

<select> / <option>

Dropdown menus

<select><option>One</option></select>

<textarea>

Multi-line text input

<textarea>Message</textarea>

<button>

Clickable button

<button>Click</button>

Media & Embeds

<video>

Embeds video content

<video src="video.mp4" controls></video>

<audio>

Embeds audio content

<audio src="sound.mp3" controls></audio>

<iframe>

Embeds external content

<iframe src="https://example.com"></iframe>

Semantic Elements

<header>

Introductory content or navigation

<header>Site header</header>

<main>

Main content area

<main>Content</main>

<section>

Thematic grouping of content

<section>Topic</section>

<article>

Independent article or content unit

<article>Blog post</article>

<aside>

Sidebar or complementary content

<aside>Extra info</aside>

<footer>

Footer content

<footer>Site footer</footer>
